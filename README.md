Little terminal game in c/c++. (for fun)
- Run's on Ubuntu 24.04.
- clone and run as so `g++ main.cpp world.cpp move.cpp item.cpp -lcurses -pthread && ./a.out`
- I moves up
- J moves left
- K moved down
- L moves right
